## Overview
This Sales Management System, built using Laravel and Tailwind CSS, is a web-based dashboard for managing key business operations.

### Key Features:
- **User Authentication & Roles**: Secure login with admin and user roles.
- **Products Module**: Add, edit, delete, and list products.
- **Sales & Invoices**: Create, view, and manage sales and invoices.
- **Reporting**: Filterable reports with export to PDF.
- **Users Management**: List and manage users and their roles (admin).
- **Settings**: Configure store information, currency, tax rate, and optional logo upload.

### Tech Stack:
- **Backend**: Laravel 10+
- **Frontend**: Blade templates and Tailwind CSS
- **Storage**: Media files via Laravel’s `public/storage`
- **PDF Export**: Using DomPDF (if implemented)
- **Version Control**: GitHub

## Project Status:
The system includes fully functional modules for products, invoices, sales, and reporting. User roles and basic settings are implemented. Next steps could include enhancing the dashboard view, real-time notifications, and multilingual support.
